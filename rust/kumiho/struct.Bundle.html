Struct Bundle 

Source 
pub struct Bundle { /* private fields */ }
Expand description

A bundle: a reserved-kind item that groups other items with a full, immutable audit trail of membership changes.

Derefs to its underlying Item, so bundle.kref, bundle.metadata, etc. are available directly.
