1//! # Kumiho Rust SDK
2//!
3//! Async Rust client for [Kumiho Cloud](https://kumiho.io) — a graph-native
4//! creative & AI asset-management system. Kumiho tracks revisions,
5//! relationships, and lineage without uploading your files ("BYO storage").
6//!
7//! This SDK mirrors the Python gold-standard SDK: a low-level [`Client`] that
8//! wraps every gRPC method, plus fluent domain types ([`Project`], [`Space`],
9//! [`Item`], [`Revision`], [`Artifact`], [`Edge`], [`Bundle`]).
10//!
11//! ## Quick start
12//!
13//! ```no_run
14//! use kumiho::{Client, EdgeType};
15//!
16//! # async fn run() -> kumiho::Result<()> {
17//! // Connect (auto-discovery from ~/.kumiho credentials, or explicit endpoint).
18//! let client = Client::connect("https://us-central.kumiho.cloud").await?;
19//!
20//! let project = client.create_project("my-vfx-project", "VFX assets").await?;
21//! let space = project.create_space("characters", None).await?;
22//! let item = space.create_item("hero", "model").await?;
23//!
24//! let revision = item.create_revision(None, 0).await?;
25//! revision.create_artifact("mesh", "/assets/hero.fbx", None).await?;
26//! revision.tag("approved").await?;
27//! # Ok(())
28//! # }
29//! ```
30//!
31//! ## Krefs
32//!
33//! A [`Kref`] is a URI uniquely identifying any object:
34//! `kref://project/space/item.kind?r=REVISION&a=ARTIFACT`.
