.. Kumiho FastAPI documentation master file

Kumiho FastAPI
==============

**Stateless, multi-tenant SaaS API for Kumiho**

This service is a thin FastAPI layer that authenticates requests (dual-token model)
and forwards operations to the Kumiho data plane.
